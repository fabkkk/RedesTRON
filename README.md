# RedesTRON
Repositório destinado aos códigos referentes ao primeiro projeto avaliativo de Redes de Computadores I, na UFPA, focado em sockets


Para testar, baixe os arquivos cliente.c, servidor.c e protocolo.h

Compile cliente.c e servidor.c com o comando gcc [NomeDoArquivo].c -o [NomeDoArquivo]

Abra duas abas de terminal, em uma delas rode o servidor, com ./servidor
Na outra, rode o cliente.c com ./cliente
O jogador 1 é representado pelo caractere "1" e se move com WASD.
